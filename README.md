# KT Nutri — Site Profissional de Nutricionista
Site profissional desenvolvido para a nutricionista Keyla Torres, com foco em acompanhamento nutricional feminino para emagrecimento sustentável, performance esportiva e saúde integral.

🌐 Acesse o site: [em breve]

## Sobre o Projeto
Site desenvolvido do zero com foco em conversão de pacientes, apresentando os serviços, planos de acompanhamento e facilitando o agendamento de consultas via WhatsApp.
O projeto foi construído com abordagem mobile first, evoluindo progressivamente para tablet e desktop, garantindo uma experiência responsiva em todos os dispositivos.

## ✦ Seções

- Header — Navbar responsiva com menu hamburguer no mobile
- Hero — Apresentação principal com CTA de agendamento
- Sobre — História e formação da nutricionista
- Serviços — Para quem é e para quem não é o acompanhamento
- Como Funciona — Etapas do acompanhamento em timeline
- Benefícios — O que está incluído no acompanhamento
- Planos — Tablist com consulta avulsa e planos trimestrais/semestrais
- Atendimento — Modalidades presencial e online
- FAQ — Perguntas frequentes com accordion
- CTA Final — Chamada para agendamento
- Footer — Informações de contato e navegação

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis customizadas
- JavaScript puro (tablist de planos)
- Google Fonts — Poppins
- Google Analytics
- Meta Pixel

## Como Rodar Localmente

- Clone o repositório:

```bash
git clone https://github.com/seuusuario/kt-nutri.git
```

- Acesse a pasta do projeto:

```bash
cd kt-nutri
```

Abra o arquivo index.html no browser ou use uma extensão como Live Server no VS Code.
